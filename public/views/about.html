Challonge TO

Challonge TO is a web app helping TOs have an easier time running brackets, inspired by Challonge Match Display.

We're still in beta, but are rolling out features as quickly as possible.

If you have any questions, comments, or feature requests, please head on over to our contact page.

Technical Details

We're open source, and the code can be found here.

Front-end: Angular, Bootstrap, hooked up to the Challonge API.

Back-end: Node, hosted on Heroku.
